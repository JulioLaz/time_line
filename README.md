# 🌍 Línea del Tiempo Global — Hitos de la Humanidad

Aplicación web interactiva para alumnos de **6.º año de Filosofía**, diseñada para dispositivos móviles.

🔗 **[Ver aplicación](https://juliolaz.github.io/time_line/)**

---

## ¿Qué es?

Una línea del tiempo interactiva que recorre los grandes acontecimientos que transformaron la humanidad, organizados en cinco ejes temáticos con profundidad filosófica en cada evento.

## Ejes temáticos

| Eje | Contenido |
|-----|-----------|
| 🟡 **Pensamiento** | Filosofía, ciencia y revoluciones del conocimiento |
| 🔴 **Poder** | Imperios, Estados y formas de gobierno |
| 🔵 **Técnica** | Inventos que transformaron la vida humana |
| 🟣 **Violencia** | Guerras y catástrofes que redefinieron el mundo |
| 🟢 **Emancipación** | Luchas por libertad y dignidad humana |

## Características

- 📱 **Mobile-first** — optimizada para celular
- 🔍 **Filtros por eje** — deslizables en la barra superior
- 📖 **Tarjetas expandibles** — tocá para ver descripción completa y perspectiva filosófica
- 👨‍🏫 **Enfoque pedagógico** — cada evento conecta el hecho histórico con pensadores y conceptos filosóficos clave

## Uso

Accedé directamente desde el navegador del celular. No requiere instalación.
